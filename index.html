способы подключения CSS к документу

CSS можно добавлять к элементу HTML тремя способами.

Внешний
когда описание стилей находится в отдельном файле, оно может быть подключено ке документу посредством элемента <link<, включённого в элемент <head<.
внутренний
когда стили описаны внутри документа, они могут быть включены в элемент <style>, который,включается в элемент <head<.
встроенный
когда стили описаны в теле документа, они могут распологаться в атрибутах style отдельного элементаю
